# LH Weekly Review

Персональный инструмент для еженедельной рефлексии и планирования — построен специально для борьбы с "имитацией дисциплины" в операционной роли.

## Что это

Два раздельных режима:

- **🟢 Ретроспектива** (пятница, 30 мин) — 8 вопросов по трём блокам: реальность недели, зоны имитации, честная оценка.
- **🟠 План недели** (понедельник, 30 мин) — 6 вопросов: главный результат, защита времени, проактивность.
- **📚 История** — все прошлые недели, с оценками и возможностью экспорта в Markdown для отчётности.

Все данные хранятся только локально в браузере (localStorage). Регулярный JSON-бэкап рекомендуется.

## Стек

Чистый static: HTML + CSS + ванильный JS. Никакого билда, никакого бэкенда. Идеально для GitHub Pages.

## Деплой на GitHub Pages

### 1. Создать репозиторий

```bash
# В папке проекта:
git init
git add .
git commit -m "Initial commit: LH Weekly Review MVP"

# Создай публичный репо на github.com (например, lh-weekly-review), затем:
git branch -M main
git remote add origin https://github.com/<твой-username>/lh-weekly-review.git
git push -u origin main
```

### 2. Включить GitHub Pages

1. Открой репо на github.com
2. Settings → Pages
3. Source: **Deploy from a branch**
4. Branch: **main** / **/ (root)**
5. Save

Через 1-2 минуты сайт будет доступен по адресу:
`https://<твой-username>.github.io/lh-weekly-review/`

### 3. (Опционально) Кастомный домен

В Settings → Pages → Custom domain. Потребует DNS-настройки.

## Структура файлов

```
├── index.html       # Главная (выбор режима)
├── retro.html       # Ретроспектива (пятница)

└── README.md
```

## Приватность

- Все данные — **только в localStorage твоего браузера**.
- Никаких запросов на сервер, никакой аналитики.
- Сайт загружает шрифт Montserrat с Google Fonts (можно отключить, если параноит).
- **Важно**: чистка кэша/cookies браузера сотрёт все данные. Делай JSON-бэкап.

## Совместимость

Работает в любом современном браузере (Chrome, Safari, Firefox, Edge). Адаптивная вёрстка для мобильных.
